# SLC Navigator - Web App

Photo-Based Indoor Navigation System for St. Lawrence College.

## Features

- Modern, minimalist UI with glassmorphism effects
- Full-screen image viewer with navigation
- Real-time pathfinding using BFS algorithm
- AI-powered navigation instructions
- Responsive design for mobile and desktop

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- shadcn/ui components
- React Router
- Axios

## Development

```bash
npm install
npm run dev
```

The app will run on `http://localhost:8080` and proxy API requests to `http://localhost:5001`.

## Build

```bash
npm run build
```
